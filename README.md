# Personio Demo

## Setup

### Prerequisites:

-   node with npm
-   yarn
-   webpack, webpack-cli

### How to setup dev env

-   Install dependencies:

`yarn`

-   Build once to copy assets

`yarn build:prod`

-   Run local server:

`yarn start`

-   Navigate to `http://localhost:3001`

### Unit tests

-   Run:

`yarn test`

-   View coverage report in `test-coverage/lcov-report/index/html`
